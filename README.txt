Description: 
	GroovesharkController is a javascript/HTML chrome extension to allow for extended use of the site http://grooveshark.com

Install: 
	NOTE: In order to run the uncompressed extension you need a developer version of chrome.  To load the unpacked extension download the root folder and place in a convenient location on your machine.  Next open chrome and go to 
chrome:///extensions
	If the developer options are not already expanded, on the top right-hand side there should be a (+) to expand the panel.  Next click load unpacked extension and navigate to the root folder you downloaded earlier and load it..You're done!  Just click the extension icon when you have http://grooveshark.com loaded and you're good to go!
